# git-commit

专业的 git 提交工作流技能。

## 功能

- 自动分析变更内容
- 判断是否需要拆分为多个 commit
- 生成符合规范的提交信息
- 支持部分提交

## 触发词

- `/commit`
- `提交代码`
- `git commit`

## 安装

```bash
npx add-skill josui/bing-ai-workflow -s git-commit
```

## 提交类型

| Type | 使用场景 |
|------|----------|
| `add` | 新增功能/组件/文件 |
| `update` | 更新/优化现有功能 |
| `fix` | 修复 bug |
| `docs` | 文档变更 |
| `style` | 代码格式调整 |
| `refactor` | 重构代码 |
| `chore` | 构建/工具/依赖更新 |
